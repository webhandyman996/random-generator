# 崑山科技大學亂數產生器

## 說明
這是一個簡單的亂數產生器網頁應用程式，用於產生指定範圍內的隨機整數。

## 功能特性
- ✅ 標題：崑山科技大學亂數產生器
- ✅ 可透過介面設定上下限值
- ✅ 只產生整數
- ✅ 響應式設計，適配各種裝置
- ✅ 支援鍵盤 Enter 鍵快速產生

## 使用方式
1. 輸入最小值（預設：1）
2. 輸入最大值（預設：100）
3. 點擊「產生亂數」按鈕或按 Enter 鍵
4. 結果會顯示在下方
